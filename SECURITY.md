# 🔐 Security Policy

## Supported Versions

We currently support the latest version of this project.

| Version | Supported |
|--------|----------|
| Latest | ✅ |

---

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

📧 Contact: contact@fivehosting.net

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

---

## Security Measures

This project includes:

- JWT Authentication
- HttpOnly Cookies
- Content Security Policy (CSP)
- Input validation (backend)
- Secure headers (XSS, Clickjacking, etc.)

---

## Best Practices

- Do not expose `.env`
- Use HTTPS in production
- Keep dependencies updated

---

## Disclaimer

This project is provided as-is. Security is continuously improved but no guarantees are made.
