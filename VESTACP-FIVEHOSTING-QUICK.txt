## 🚀 3 étapes pour déployer (30 min)

### ÉTAPE 1: SSH + Code (5 min)

```bash
# SSH
ssh admin@185.171.202.132

# Créer dossier app
mkdir -p /home/admin/web/heurejo.fivehosting.net/calcul-heure
cd /home/admin/web/heurejo.fivehosting.net/calcul-heure

# Télécharger code
git clone https://github.com/votre_user/calcul-heure .

# Ou via SCP depuis votre PC:
# scp -r ./* admin@185.171.202.132:/home/admin/web/heurejo.fivehosting.net/calcul-heure/

# Vérifier
ls -la
```

### ÉTAPE 2: Installation + Config (5 min)

```bash
# npm install
npm install

# Générer JWT_SECRET
openssl rand -base64 32
# Copier le résultat

# Éditer .env
nano .env
# PORT=3001
# JWT_SECRET=COLLER_LE_RÉSULTAT
# NODE_ENV=production
# Ctrl+O, Enter, Ctrl+X

# Tester localement
npm start
# Ctrl+C pour arrêter
```

### ÉTAPE 3: PM2 + Nginx + Tests (10 min)

```bash
# PM2
npm install -g pm2

# Config PM2
nano ecosystem.config.js
# Copier depuis VESTACP-FIVEHOSTING.md (Étape 6)
# Ctrl+O, Enter, Ctrl+X

# Démarrer
pm2 start ecosystem.config.js
pm2 status
# Résultat: online ✅

# Config Nginx
sudo nano /home/admin/conf/web/nginx.heurejo.fivehosting.net.conf
# Copier contenu de nginx-config-FIVEHOSTING-FINAL.conf
# Adapter IP et domaine
# Ctrl+O, Enter, Ctrl+X

# Tester Nginx
sudo nginx -t

# Recharger
sudo systemctl reload nginx

# Vérifier ports
sudo netstat -tlnp | grep 3001
sudo netstat -tlnp | grep nginx

# HTTPS (VestaCP Panel)
# https://185.171.202.132:8083/ → Domaines → SSL → Activer Let's Encrypt

# ✅ TESTER
# https://heurejo.fivehosting.net
```
