## 🔥 3 étapes pour déployer

### ÉTAPE 1: SSH + Application

```bash
# 1. Connexion SSH
ssh heurejo@ssh-heurejo.fivehosting.net

# 2. Créer dossier et télécharger
mkdir -p ~/app/calcul-heure
cd ~/app/calcul-heure

# 3. Via Git (recommandé)
git clone https://github.com/votre_user/calcul-heure .

# OU via SCP depuis votre PC (PowerShell)
# scp -P 22 -r ./* heurejo@ssh-heurejo.fivehosting.net:~/app/calcul-heure/

# 4. Vérifier
ls -la
# Vous devez voir: package.json, server.js, .env
```

### ÉTAPE 2: Installation + Configuration

```bash
# 1. Dépendances
npm install

# 2. Créer JWT_SECRET fort
openssl rand -base64 32
# Copier le résultat

# 3. Éditer .env
nano .env
# Remplacer JWT_SECRET par le résultat d'openssl
# S'assurer: PORT=3001

# 4. Sauvegarder (Ctrl+O, Enter, Ctrl+X)
```

### ÉTAPE 3: PM2 + Nginx + Tests

```bash
# 1. PM2 installation globale
npm install -g pm2

# 2. Configuration PM2
nano ecosystem.config.js
# Copier le contenu du FIVEHOSTING-GUIDE.md (Étape 11)
# Ctrl+O, Enter, Ctrl+X

# 3. Démarrer
pm2 start ecosystem.config.js
pm2 status

# 4. Apache proxy (le plus courant chez FiveHosting)
cd ~/public_html
nano .htaccess
# Copier le contenu du FIVEHOSTING-GUIDE.md (Étape 13 - Option A)
# Ctrl+O, Enter, Ctrl+X

# 5. Tester HTTPS
# Ouvrir navigateur: https://heurejo.fivehosting.net
# ✅ Vous devez voir la page de login
```

## 🎯 Après le déploiement

```bash

# Sauvegarder la base
cp ~/app/calcul-heure/database.db ~/app/calcul-heure/backups/database_$(date +%Y%m%d).db

git pull
npm install
```

## 🆘 Erreurs courantes

**Port 3001 occupé?**
```bash
lsof -i :3001
kill -9 <PID>
```

**PM2 pas trouvé?**
```bash
npm install -g pm2
```

**App crash?**
```bash
pm2 logs calcul-heure
# Vérifier le JWT_SECRET dans .env
```
