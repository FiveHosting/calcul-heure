
## 🚀 Commandes rapides VestaCP

### Première fois
```bash
ssh root@ip
bash deploy-vestacp.sh         # 5-10 min
sudo nano /etc/nginx/sites-available/calcul-heure
sudo nginx -t && sudo systemctl restart nginx
sudo certbot certonly -d votre_domaine.com
```

### Maintenance quotidienne
```bash
pm2 status                      # OK?
pm2 logs calcul-heure --lines 20   # Erreurs?
```

### En cas de problème
```bash
pm2 logs calcul-heure           # Voir erreurs
pm2 restart calcul-heure        # Redémarrer
sudo systemctl restart nginx    # Relancer Nginx
rm database.db && pm2 restart   # Reset DB
```
